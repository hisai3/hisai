mysql相关操作

发表于2021-07-22|更新于2023-01-31|webpack

一些最重要的 SQL 命令

  • SELECT - 从数据库中提取数据
  • UPDATE - 更新数据库中的数据
  • DELETE - 从数据库中删除数据
  • INSERT INTO - 向数据库中插入新数据
  • CREATE DATABASE - 创建新数据库
  • ALTER DATABASE - 修改数据库
  • CREATE TABLE - 创建新表
  • ALTER TABLE - 变更（改变）数据库表
  • DROP TABLE - 删除表
  • CREATE INDEX - 创建索引（搜索键）
  • DROP INDEX - 删除索引

数据库

备份数据库

  • mysqldump -u root 库名>xxx.data

查询 show

  • 所有数据库名： show databases;
  • 当前数据库：show database();

添加 create database

  • create database name

删除 drop database

  • drop database name

修改 ALTER DATABAS

  • mysql> ALTER DATABASE test_db
    -> DEFAULT CHARACTER SET gb2312
    -> DEFAULT COLLATE gb2312_chinese_ci;

表

查询 (主要是对列的查询) show

  • 所有表名：show tables from tabl_name;

添加 create

  • create table MYTABLE (列名 类型 是否为空 默认值， …); // CREATE TABLE MYTABLE (name VARCHAR(20), sex CHAR(1));

删除 drop

  • drop table MYTABLE ;

修改 对列的修改 alter table

  • 清空表中记录：delete from MYTABLE ;
  • 清空表中指定条件记录：delete from MYTABLE where course_id=4;
  • 修改表名：alter table tabl_name1 rename to tabl_name2;

列

查询 show describe/desc

  • 所有列：show columns in MYTABLE ;
  • 所有列：describe/desc MYTABLE ;

添加 add

列包含 col_name type isnull default —> new1 VARCHAR(20) DEFAULT NULL

alter table 语句

增加一个字段，列名为new1， 类型为varchar(20)，默认为空

  • alter table MYTABLE add COLUMN new1 VARCHAR(20) DEFAULT NULL;

增加一个字段，列名为new2， 类型为varchar(20)，默认不能为空

  • alter table user add COLUMN new2 VARCHAR(20) NOT NULL;

删除 drop

  • alter table MYTABLE drop column new1;

修改 change / modify

  • 修改列名：alter table col_name1 change col_name2 (列包含);
  • 修改列类型：alter table t_book modify name varchar(22);

行

查询 select … from

  • 所有列的数据：select * from tabl_name;
  • 指定列的数据：select spec_column_name from tabl_name;

添加 insert into … values

  • insert into MYTABLE values (“hyq”,”M”); 插入完整行 values 值需要跟列的数量一致

删除 delete

  • 清空表中指定条件记录：delete from MYTABLE where course_id=4;

修改 update

  • 修改指定列的行数据：update MYTABLE set sex=”f” where name=’hyq’;
